# Rust Inventory System

This repository contains a simple inventory system written in Rust.

## Features
- Add items with inventory capacity check
- Remove items by index
- Calculate total inventory value
- Uses enum for item types (Weapon, Armor, Potion)

## How to Run
1. Open Rust Playground
2. Paste the code from `src/main.rs`
3. Click Run
